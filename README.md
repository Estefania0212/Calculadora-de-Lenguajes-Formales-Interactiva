# 🧮 Calculadora de Lenguajes Formales Interactiva

## 📌 Descripción

Este proyecto consiste en el desarrollo de una aplicación web interactiva orientada a facilitar el cálculo y la comprensión de operaciones con alfabetos, cadenas y lenguajes formales en el área de la Teoría de la Computación.

La aplicación permite a los estudiantes realizar operaciones de forma automatizada mediante una interfaz visual e intuitiva, reduciendo la complejidad de los cálculos manuales y mejorando el aprendizaje de estos conceptos abstractos.

---

## ❗ Problema

Actualmente, los estudiantes presentan dificultades para comprender conceptos como alfabetos, cadenas y lenguajes, debido a:

- Explicaciones altamente abstractas  
- Falta de herramientas visuales  
- Errores frecuentes en cálculos manuales  
- Bajo vínculo entre la teoría y la práctica  

Esto genera un aprendizaje lento y poco efectivo.

---

## ❓ Pregunta de Investigación

¿Cómo influye el desarrollo de una aplicación web en la mejora del cálculo y comprensión de operaciones con alfabetos, cadenas y lenguajes en la enseñanza de la Teoría de la Computación?

---

## 🎯 Objetivo General

Desarrollar una aplicación web interactiva que facilite el cálculo y la comprensión de operaciones con alfabetos, cadenas y lenguajes, mediante herramientas visuales y prácticas interactivas, con el fin de fortalecer el aprendizaje de los estudiantes.

---

## 🎯 Objetivos Específicos

- Identificar los requerimientos funcionales y no funcionales de la aplicación.  
- Diseñar una interfaz gráfica intuitiva mediante wireframes.  
- Desarrollar la aplicación utilizando PHP, XAMPP y herramientas web.  
- Implementar funcionalidades para operaciones con alfabetos, cadenas y lenguajes.  
- Fortalecer el aprendizaje mediante el uso de una calculadora interactiva.  

---

## 💡 Justificación

El uso de herramientas tecnológicas permite mejorar la comprensión de conceptos abstractos. Esta aplicación busca:

- Reducir errores en cálculos manuales  
- Automatizar procesos complejos  
- Disminuir la carga cognitiva  
- Promover un aprendizaje activo  

Beneficiando principalmente a estudiantes de ingeniería y áreas afines.
